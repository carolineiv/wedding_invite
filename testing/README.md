# Тестирование свадебного сайта-приглашения

Этот раздел содержит результаты ручного тестирования сайта-приглашения, который был разработан для реального мероприятия.

## Что тестировалось

- **Функциональность** -добавление гостей, отправка в Telegram, работа музыки
- **Регрессия** - проверка, что исправления не сломали старый функционал
- **Юзабилити** - удобство интерфейса для пользователя
- **Конфигурация** - работа в разных браузерах и на разных устройствах

## Структура папки

- `test-cases.md` - список тест-кейсов
- `bug-reports.md` - найденные и исправленные баги
- `checklist.md` - быстрая проверка всех функций
- `test-report.md` - итоговый отчёт
- `assets/` - скриншоты, подтверждающие работу

## Вывод

Все основные функции работают корректно. Критических багов не обнаружено. Проект готов к использованию.
